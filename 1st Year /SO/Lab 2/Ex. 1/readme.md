**[wc #2]** (max 0.25p)

Să se scrie comanda ce afișează câte linii de text conține fișierul /etc/mtab, precum și lungimea celei mai lungi linii de text din acel fișier.

<sub>(Indicație: folosiți comanda wc.) </sub>

<br> 

**Rezolvare:**

```terminal
wc -l -L /etc/mtab
```
